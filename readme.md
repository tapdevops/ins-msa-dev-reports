# Foobar

Untuk menyimpan data-data report yang dijalankan dari cron.

## Installation

Instalasi project :

```bash
git clone 
```

## Usage

```python
import foobar

foobar.pluralize('word') # returns 'words'
foobar.pluralize('goose') # returns 'geese'
foobar.singularize('phenomena') # returns 'phenomenon'
```
